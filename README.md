# Mathruseva Foundation - Integrated Volunteer & Impact Management System

## Project Overview
A web-based system for coordinating NGO volunteers, managing camps/events, tracking donations, and generating impact analytics for Mathruseva Foundation.

## Tech Stack
- **Frontend**: HTML, CSS, Bootstrap, JavaScript
- **Backend**: Python Flask
- **Database**: MySQL
- **Analytics**: Python Pandas, Chart.js
- **PDF Generation**: Python libraries

## User Roles
- **Admin (NGO Coordinator)**: Full control
- **Volunteers**: Organizers, Helpers, Support Staff
- **Doctors**: External (no system access)
- **Students/Beneficiaries: Counted in analytics only

## Modules
1. Volunteer Management
2. Camp & Event Management
3. Medical Camp Summary Data
4. Donation & Supplies Tracking
5. Impact Analytics Dashboard
6. PDF Impact Reports

## Installation Instructions
1. Install Python 3.8+
2. Install MySQL Community Server and Workbench
3. Create database and tables
4. Install Python dependencies
5. Run Flask application

## Database Schema
- volunteers
- camps
- camp_volunteers
- medical_summary
- donations
